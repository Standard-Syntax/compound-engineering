# Local Development Commands

These commands are local development shims for the plugin skills in `plugins/compound-engineering/skills/`. They let contributors run the workflow commands when working on the repo without installing the plugin.

**The plugin skills are the canonical source of truth.** If a command's behavior needs to change, the change belongs in `plugins/compound-engineering/skills/<skill>/SKILL.md` and should be mirrored to the corresponding root command afterward.

| Root command | Plugin skill |
|---|---|
| `ce:work.md` | `plugins/compound-engineering/skills/ce-work/SKILL.md` |
| `ce:review.md` | `plugins/compound-engineering/skills/ce-review/SKILL.md` |
| `ce:compound.md` | `plugins/compound-engineering/skills/ce-compound/SKILL.md` |
| `ce:calibrate.md` | `plugins/compound-engineering/skills/calibrate/SKILL.md` |
| `ce:scaffold.md` | `plugins/compound-engineering/skills/scaffold/SKILL.md` |
| `ce:retrospect.md` | `plugins/compound-engineering/skills/retrospect/SKILL.md` |
| `triage-prs.md` | `plugins/compound-engineering/skills/triage/SKILL.md` |
